GMOJ 4722.跳楼机

发表于 2021-03-18 | 更新于: 2021-04-08 | 分类于 模拟赛题目 |

题目原文

DJL为了避免成为一只咸鱼，来找srwudi学习压代码的技巧。
Srwudi的家是一幢h层的摩天大楼。由于前来学习的蒟蒻越来越多，srwudi改造了一个跳楼机，使得访客可以更方便的上楼。
经过改造，srwudi的跳楼机可以采用以下四种方式移动：
1、向上移动x层；
2、向上移动y层；
3、向上移动z层；
4、回到第一层。
一个月黑风高的大中午，DJL来到了srwudi的家，现在他在srwudi家的第一层，碰巧跳楼机也在第一层。DJL想知道，他可以乘坐跳楼机前往的楼层数。

简化版

输入h,x,y,z。

现在有三种操作：

1.向前走x格。

2.向前走y格。

3.向前走z格。

而你现在在第一格。

现在问：他可以到达的并且小于等于h的格数。

样例输入

15
4 7 9

样例输出

9

样例解释

可以到达的：1，5=1+4，8=1+7，9=1+4+4,10=1+9,12=1+7+4,13=1+4+4+4,14=1+4+9,15=1+7+7

数据范围

20%的数据，1≤h, x, y, z≤100；
40%的数据，1≤h, x, y, z≤10^5；
100%的数据，1≤h≤10^18，1≤x, y, z≤10^5

题解

40%

直接完全背包莽过去

时间复杂度O（h）

空间复杂度O（h）

代码
#include<iostream>
#include<cstdio>
#include<cstring>
using namespace std;
int
 n,a,b,c,ans;
bool
 dp[50000001];
int main()
{
    scanf("%d%d%d%d",&n,&a,&b,&c);
    dp[1]=1;
    for(int i=1;i<=n;i++)
        if(dp[i]) dp[i+a]=dp[i+b]=dp[i+c]=1,ans++;
    printf("%d",ans);
}
100%

设a[i]为b%x==i时b的最小值

b的定义为只用y和z能走到的格子

那么答案就是:

问题就转换为如何求a[i]的值

很容易就可以得到两个转移

那么现在就有两种方法了

方法1:同余最短路

将上面的公式看作为

i到(i+y) mod x的边权值为a[i]+y

i到(i+z) mod x的边权值为a[i]+z

然后连边

那么问题就转换成最短路问题

spfa就解决了

程序
#include<iostream>
#include<cstdio>
#include<cstring>
using namespace std;
long long
 n,m,x,y,z,d[10000001],bian[1000001],tot=1,q[1000001],hand,tail=1,ans;
bool
 a[10000001];
struct s
{
    int end,next,v;
}l[1000001];
void add(int x,int y,int v)
{
    l[tot].end=y;
    l[tot].next=bian[x];
    l[tot].v=v;
    bian[x]=tot++;
}
int main()
{
    memset(d,100,sizeof(d));
    scanf("%lld%lld%lld%lld",&n,&x,&y,&z);
    if(x>y) swap(x,y);
    if(y>z) swap(y,z);
    if(x>z) swap(x,z);
    if(x==1)
    {
        printf("%lld",n);
        return 0;
    }
    for(int i=0;i<x;i++)
    {
        add(i,(i+y)%x,y);
        add(i,(i+z)%x,z);
    }
    q[1]=1;
    d[1]=1;
    while(hand<tail)
    {
        int o=q[++hand];
        a[o]=0;
        for(int i=bian[o];i!=0;i=l[i].next)
        {
            if(d[l[i].end]>d[o]+l[i].v)
            {
                d[l[i].end]=d[o]+l[i].v;
                if(a[l[i].end]==0)
                {
                    q[++tail]=l[i].end;
                    a[l[i].end]=1;
                }
            }
        }
    }
      for(int i=0;i<x;i++)
    {
        if(n>=d[i])
        ans=ans+(n-d[i])/x+1;
    }
    printf("%lld",ans);
}

时间复杂度约o(x)

常数略大，约100ms

方法2:BFS

由i拓展到(i+y) mod x和(i+z) mod x

条件为+y、+z是否小于原来的值

如果是则赋值

判断是否在队列里

如果没有就放进去

代码较短，所以不贴出来了

时间复杂度约x)

常数较小，约30~50ms

(这得看你的代码常数处理丑不丑陋了)
