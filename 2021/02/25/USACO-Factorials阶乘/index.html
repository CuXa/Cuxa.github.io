USACO Factorials阶乘

发表于 2021-02-25 分类于 USACO

【USACO】 Factorials阶乘

题目大意

给定一个数n，求n！的最后面的非零位

样例输入

7

样例输出

4

样例解释

1 * 2 * 3 * 4 * 5 * 6 * 7 = 5040

最后面的非零位是4

输出4

数据范围

n<=4,200

题解

经典水题，与GDKOI的一题很像，也是求最后面的非零位，做法也差不多。先把n！看成1 * 2 * 3 * …* (n-1) * n然后分解质因数，并且把1~n每个质因数的个位数的个数存起来，再将2与5抵消，判断是否有剩5，如果有，那么直接输出5，没有就从2到7将剩余的乘起来，变为2^n[2] * 3^n[3]* 5^n[5] * 7^n[7] ,乘的时候取个位就好了，最后再输出，但其实最后一步可以改进，用规律来算，但是范围太小，不值得。

AC程序

#include<iostream>
#include<cstdio>
#include<cstring>
using namespace std;
int
 n,p[100001],k,ans=1;
int main()
{
    scanf("%d",&n);
    for(int i=2;i<=n;i++)
    {
        k=i;
        int j=2;
        while(k!=1)
        {
            while(k%j==0) p[j%10]++,k/=j;
            j++;
        }
    }
    int o=p[5];
    p[5]=p[5]-min(o,p[2]),p[2]=p[2]-min(o,p[2]);
    if(p[5]>0)
        printf("5");
    else
        for(int i=2;i<=n;i++)
            if(p[i]>0)
                for(int j=1;j<=p[i];j++)
                    ans=ans*i%10;
    printf("%d",ans);
}
